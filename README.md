Boss直聘投递统计助手 —— 一款自动统计「招呼数」和「简历交换数」的 Tampermonkey 油猴脚本。 支持按今天/昨天/更早多维度统计，数据持久化存储，面板可拖拽、可折叠，刷新不丢失。 让每一次沟通都有数据可查，求职进度一目了然。
## 项目简介

在 Boss 直聘上与大量 HR 沟通时，很难手动统计自己每天发了多少“招呼”、成功交换了多少份简历。本脚本直接在聊天列表页注入一个**悬浮统计面板**，自动扫描对话记录，按“今天 / 昨天 / 更早”三个维度统计招呼数和简历交换数，所有数据持久化存储，刷新页面或重启浏览器都不会丢失。

**解决的痛点**：
- 不知道当天已主动沟通多少家公司
- 无法快速统计有效简历交换数量
- 手动记录繁琐且容易遗漏
- 数据无法跨日追溯

## 功能亮点

- ✅ **自动扫描** – 每 0.8 秒自动检测聊天列表变化，无需手动刷新
- ✅ **智能区分** – 识别“今天 / 昨天 / 更早”的消息时间，分别统计
- ✅ **简历交换识别** – 自动检测是否开启了“有交换”筛选，单独计入简历数
- ✅ **持久化存储** – 使用 `localStorage` 保存数据，关闭浏览器也不丢失
- ✅ **拖拽移动** – 面板可任意拖拽到屏幕边缘，位置自动记忆
- ✅ **折叠/展开** – 点击 `−` 可折叠为圆形图标，不遮挡聊天区域
- ✅ **视觉反馈** – 数据变化时有黄色闪烁动画，实时感知统计更新
- ✅ **一键清空** – 提供清空按钮，方便重新开始统计

## 安装方式

### 前置条件
- 浏览器需安装 **Tampermonkey** 或 **Violentmonkey** 扩展

### 安装步骤
1. 点击以下链接，安装 `.user.js` 脚本文件：
   https://github.com/fengpr/Boss-Zhipin-Statistics/blob/master/Boss%E7%9B%B4%E8%81%98%E6%8A%95%E9%80%92%E7%BB%9F%E8%AE%A1/boss-stats.user.js
2. Tampermonkey 会自动识别并弹出安装页面，点击「安装」即可

3. 打开 Boss 直聘「沟通」页面：  
`https://www.zhipin.com/web/geek/chat*`

4. 页面右下角会出现统计面板，立即开始自动统计

## 使用说明

### 基本操作
| 操作 | 效果 |
|------|------|
| 拖拽面板标题栏 | 移动面板位置（自动保存） |
| 点击 `−` 按钮 | 折叠为圆形图标 📊 |
| 点击圆形图标 | 展开面板 |
| 点击「清空历史」 | 清除所有统计数据（需确认） |

### 统计逻辑
- **招呼数**：每个聊天列表项（li[role="listitem"]）自动计入一次招呼，同一家公司只统计一次
- **简历数**：仅在页面筛选条件为「有交换」时自动累加
- **时间分类**：
- 消息含 `:` → 今天
- 消息含 `昨天` → 昨天
- 其他 → 更早
